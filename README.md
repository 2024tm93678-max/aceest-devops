# ACEest Fitness & Gym - DevOps CI/CD Pipeline

## 📌 Project Overview
This project demonstrates the implementation of a complete DevOps CI/CD pipeline for the ACEest Fitness & Gym application. The objective is to automate the build, testing, and deployment process using modern DevOps tools.

---

## 🛠️ Technologies Used
- Python (Flask)
- Git & GitHub
- Docker
- GitHub Actions
- Jenkins
- Pytest

---

## 🚀 Application Description
A simple Flask-based web application that displays:

"Welcome to ACEest Fitness & Gym 💪"

This application serves as the base system for implementing DevOps practices.

---

## 🖥️ Additional Application Modules (Tkinter Versions)

The project also includes multiple versions of a desktop-based ACEest Fitness application developed using Tkinter.

These files demonstrate the progressive development of various features such as:
- Dashboard interface
- Data handling modules
- Visualization using matplotlib
- Report generation using FPDF

Note: These modules are GUI-based and are not integrated into the CI/CD pipeline, as the pipeline is designed for the Flask web application.
